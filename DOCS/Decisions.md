# Project Decisions
This document records important technical and business decisions.

## Decision 001
Date:
July 13, 2026

Decision:
Use Flutter for mobile application development.

Reason:
Flutter allows one codebase to support Android and iOS.

## Decision 002
Date:
July 13, 2026

Decision:
Use Python FastAPI for backend development.

Reason:
Python provides flexibility, scalability, and rapid development.

## Decision 003
Date:
July 13, 2026

Decision:
Use a local-first data approach.

Reason:
Users should be able to use the app without internet and synchronize later.

## Decision 004
Date:
July 13, 2026

Decision:
User data will not be stored in GitHub.

Reason:
GitHub is for source code management, not private user data storage.
